### RPy2 Docker

![Docker Cloud Build Status](https://img.shields.io/docker/cloud/build/achillesrasquinha/rpy2.svg)

> Alpine-based Docker Image for rpy2

### Usage

```
$ docker run \
    --rm \
    -it \
    achillesrasquinha/rpy2 \
    python3 -c "import rpy2; print(rpy2.__version__)"
```
